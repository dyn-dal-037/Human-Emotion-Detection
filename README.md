# Human Emotion Detection

This repository implements a **human emotion detection system** using deep learning on facial images. The project focuses on classifying basic human emotions from facial expressions using **Convolutional Neural Networks (CNNs)** and **transfer learning**, with an emphasis on model interpretability.

---

## Overview

The system takes facial images as input and predicts the corresponding emotional state. In addition to classification, the project explores **model interpretability** using **Grad-CAM** to visualize which facial regions contribute most to the modelâ€™s predictions.

This work serves as a practical study of **computer vision pipelines**, **deep learning training workflows**, and **explainable AI techniques**.

---

## Key Features

- Facial emotion classification using CNNs  
- Transfer learning with pretrained models  
- Model interpretability using Grad-CAM  
- Dataset preprocessing and augmentation  
- Performance evaluation and error analysis  

---

## Technologies Used

- Python  
- TensorFlow / Keras  
- Convolutional Neural Networks (CNNs)
